# Examples

The single example project contains multiple `main.dart` files:
https://github.com/akvelon/flutter-code-editor/tree/main/example

Run any of them for the specific example app.

## 01.minimal

![01.minimal](https://raw.githubusercontent.com/akvelon/flutter-code-editor/main/example/images/01.minimal.png)

This is the minimal example that uses `CodeController` with an ordinary `TextField`.

Run this:
https://github.com/akvelon/flutter-code-editor/tree/main/example/lib/01.minimal

## 02.code_field

![02.code_field](https://raw.githubusercontent.com/akvelon/flutter-code-editor/main/example/images/02.code_field.png)

`CodeField` widget adds a gutter, code folding, and basic autocompletion.

Run this:
https://github.com/akvelon/flutter-code-editor/tree/main/example/lib/02.code_field

## 03.change_language_theme

![03.change_language_theme](https://raw.githubusercontent.com/akvelon/flutter-code-editor/main/example/images/03.change_language_theme.png)

This example shows how to change language and theme dynamically.

Run this:
https://github.com/akvelon/flutter-code-editor/tree/main/example/lib/03.change_language_theme
